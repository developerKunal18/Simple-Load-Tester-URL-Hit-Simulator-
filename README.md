# Simple Load Tester

A Python script that simulates multiple requests to a URL.

## Description
This project sends repeated HTTP requests to a given URL and tracks success and failure counts.

## Features
- Simulates load
- Tracks response success
- Useful for testing APIs

## Technologies Used
- Python
- urllib module
- time module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python load_tester.py

4. Enter URL and request count

## Example

Output:
Success: 8  
Failed: 2
